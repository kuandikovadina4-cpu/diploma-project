# Diploma Project

## Жоба тақырыбы
Пайдаланушының мәліметтерді қабылдаудағы визуализация түрлерінің тиімділігін зерттеу

## Авторы
- Куандыкова Дина
- Аралбай Қанағат

## Жоба сипаттамасы
Бұл дипломдық жоба пайдаланушылардың ақпаратты қабылдауына визуализация түрлерінің әсерін зерттеуге арналған. 
Жоба барысында әртүрлі визуализация әдістері салыстырылып, олардың тиімділігі талданады.

## Репозиторий құрылымы
- `introduction` – кіріспе бөлімі  
- `Chapter 1` – теориялық бөлім  
- `Process Optimization` – жобалық бөлім  
- `Security & Integrity` – техникалық талаптар  
- `User Guide` – пайдаланушы нұсқаулығы  

## Қолданылған технологиялар
- R / Shiny
- GitHub
- Data Visualization құралдары

## Бірлескен жұмыс
Жоба GitHub платформасында командалық түрде орындалды:
- файлдар бірге өңделді
- commit және өзгерістер тарихы сақталды
- аудит және рецензия жүргізілді

## Қорытынды
Бұл жоба визуализацияның пайдаланушы қабылдауына әсерін көрсетіп, тиімді әдістерді анықтауға мүмкіндік береді.
